# 项目简介

## 核心功能
- 用户和管理员模块
- 实验室人员信息管理
- 供应商信息管理
- 实验室类型管理
- 实验室信息管理
- 耗材分类管理
- 耗材信息管理
- 耗材入库管理
- 耗材调整管理
- 采购申请管理
- 耗材借用申请管理
- 耗材归还管理
- 耗材报废管理
- 耗材维修申请管理
- 耗材维修反馈管理
- 收支信息管理

## 适用场景
- 实验室耗材管理
- 耗材采购和库存管理
- 耗材借用和归还管理
- 耗材报废和维修管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
